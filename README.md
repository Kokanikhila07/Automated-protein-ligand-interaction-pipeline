# 🔬 Automated Multi-Ligand Protein–Ligand Docking & Interaction Analysis Pipeline

> 🚀 An end-to-end automated Structure-Based Drug Design (SBDD) workflow that performs multi-ligand docking, interaction profiling, and visualization with organized, reproducible output generation.

---

## 📌 Project Overview

This project implements a fully automated computational pipeline that:

* Accepts a target protein structure (`.pdb`)
* Accepts multiple ligand files (`.sdf`)
* Performs molecular docking
* Extracts best binding poses
* Generates protein–ligand complexes
* Performs detailed interaction analysis
* Automatically generates:

  * 📄 Interaction reports (XML & TXT)
  * 🖼️ 2D Hydrogen Bond Network
  * 🌐 Interactive 3D Visualization (HTML)
  * 📁 Structured result folders for each ligand

The pipeline is scalable and designed for **multi-ligand virtual screening**.

# ⚙️ Features

## 1️⃣ Automated Multi-Ligand Docking

* Loops through all `.sdf` files automatically
* Converts receptor and ligands to `.pdbqt` format
* Performs docking using AutoDock Vina
* Extracts best binding pose (Pose 1)
* Processes ligands sequentially without manual intervention

---

## 2️⃣ Complex Generation

* Merges protein + docked ligand
* Generates protein–ligand complex structure
* Performs bond perception correction using RDKit
* Prepares protonated and cleaned complex
* Automatically prepares structure for interaction profiling

---

## 3️⃣ Interaction Analysis

Interaction profiling is performed using PLIP.

Detected interactions include:

* Hydrogen bonds
* Hydrophobic interactions
* Salt bridges
* π–π stacking
* Water bridges

Generated outputs:

* `report.xml` – Machine-readable interaction data
* `report.txt` – Human-readable interaction summary

---

## 4️⃣ Custom 2D Hydrogen Bond Network

* Parses PLIP XML output
* Extracts hydrogen bond residues
* Generates circular interaction network using NetworkX
* Ligand represented as **central yellow node**
* Residues represented as **green-bordered square nodes**
* Hydrogen bonds shown as **blue dashed edges**
* Exported as **high-resolution PNG (300 dpi)**

---

## 5️⃣ Interactive 3D Visualization

* Generated using py3Dmol
* Cartoon protein representation
* Stick & sphere ligand model
* Van der Waals surface rendering
* Auto-zoom to binding site
* Exported as interactive `.html` file

# 🚀 How to Run

### 1️⃣ Install Dependencies

```bash
conda install -c conda-forge openbabel plip
pip install rdkit networkx matplotlib py3Dmol meeko
```

Install AutoDock Vina separately.

---

### 2️⃣ Modify Docking Grid

Edit grid parameters in:

```
scripts/dock_run_multiple.sh
```

Adjust:

```
CENTER_X
CENTER_Y
CENTER_Z
SIZE_X
SIZE_Y
SIZE_Z
```

---

### 3️⃣ Execute Pipeline

```bash
chmod +x scripts/dock_run_multiple.sh
./scripts/dock_run_multiple.sh
```
